Theory A. CAESAR METHOD
It is a substitution cipher in which the alphabet in the plain text is shifted by a fixed number down the alphabet.
The value of key is 3.
Formula used is:
Cipher = (p+k) mod 26, where p=plain text, k = key
Plain = (c-k) mod 26, where c=Cipher text, k = key

Advantages of the method includes:
1. Easy to encrypt a plain text
2. Short key
3. No complexity

Disadvantages of the method includes:
1. Minimum Security
2. Key value is fixed


B. ADDITIVE METHOD
It is a substitution cipher in which the alphabet in the plain text is shifted by a fixed number down the alphabet.
The key is variable in this method.
Formula used is:
Cipher = (p+k) mod 26, where p=plain text, k = key
Plain = (p-k) mod 26, where p=plain text, k = key

Advantages of the method includes:
1. Easy to encrypt a plain text
2. Variable key
3. Better security

Disadvantages of the method includes:
1. Minimum Security
2. Single alphabet replacement
