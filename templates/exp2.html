Objective

To understand Encryption Decryption using Multiplicative ciphering method.


Theory

Definition:
: It is a substitution cipher in which the alphabet in the plain text is shifted by a number down the alphabet. The value of key is variable.
Formula:
k = Key
To encrypt -> (p*k) mod 26, Where p=plain text
To decrypt -> (c*k^(-1)) mod 26, Where c=Cipher text
Advantages:
1. Increased Security
2. Custom key
3. Increased complexity
Disadvantages:
1. Limited Key Space
2. Difficult with Non-Coprime Numbers
