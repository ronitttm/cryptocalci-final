Theory

A. Caeser Method

Definition:
It is a substitution cipher in which the alphabet in the plain text is shifted by a fixed number down the alphabet. The value of key is 3.
Formula:
k = Key
To encrypt - (p+k) mod 26, Where p=plain text
To decrypt - (c-k) mod 26, Where c=Cipher text
Advantages:
1. Easy to encrypt a plain text
2. Short key
3. No complexity
Disadvantages:
1. Minimum Security
2. Key value is fixed



B. Additive Method

Definition:
It is a substitution cipher in which the alphabet in the plain text is shifted by a fixed number down the alphabet. The key is variable in this method.
Formula:
k = Key
To encrypt - (p+k) mod 26, Where p=plain text
To decrypt - (c-k) mod 26, Where c=Cipher text
Advantages:
1. Easy to encrypt a plain text
2. Variable key
3. Better Secirity
Disadvantages:
1. Minimum Security
2. Single Alphabet Replacement
