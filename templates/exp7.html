A. Reconnaissance Tools

1.Ipvoid :
IPVoid is a free service to scan an IP address with multiple IP. It provides useful tools to troubleshoot network problems and to help in gathering information about IP addresses.

2.DNSdumpster :
DNSdumpster is an online passive scanning tool to obtain information about domains, block addresses, emails, and all kind of information DNS related. It gives us information about MX records, TXT records and Host records

3.Whois :
WHOIS is widely used for querying databases that store the registered users or assignees of an internet resource, such as a domain-name, an IP address block or an autonomous system, but is also used for a wider range of other information

4.Smallseotools :
SmallSeoTools.com collects and uses website information to operate its website analysis tool and deliver the services you have requested. This information can include: IP address, domain names, estimated visitors, in-site and off-site SEO analysis, usability, access times and referring website addresses.
